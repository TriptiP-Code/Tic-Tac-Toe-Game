## About

Tic-Tac-Toe is a classic two-player game where players take turns marking a 3Ã—3 grid, trying to get three of their marks in a row (horizontally, vertically, or diagonally). This repository contains a basic, interactive version of that game that runs entirely in the browser.

---

## Features

- Two-player mode (X and O)  
- Turn-based interaction  
- Real-time win / draw detection  
- Responsive UI that works on desktop and mobile  
- Simple, clean design
